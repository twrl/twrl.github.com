Operating System Development

To amuse myself while I wait for the new term to start in a few weeks time, I’ve turned to a very old friend - osdev.

When I was a teenager I dabbled in Operatng Systems quite a lot. I had some great vision of how a new operating system should be, but of course I lacked both the patience and the knowledge to actually bring it about.

I’m somewhat older and wiser now, so maybe I’ll get a little further than printing “hello world”. At the moment, I’m just trying out some ideas in a very simple kernel or two, but I think I’ll sketch out a proper design and go for it.

Meanwhile, here’s a few general osdev resources for anyone interested:

  • osdev.org wiki
  • Bona Fide OS Developer
  • The Operating System Resource Centre
  • How to Write an Operating System
  • BrokenThorn OS Development
  • and there are many more out there
