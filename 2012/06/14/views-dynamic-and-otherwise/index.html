Views, dynamic and otherwise

Thursday 14 June, 2012

Blog, Blogger, Dynamic Views

I’ve changed theme on this blog. I was using Dynamic Views, and now I’m using a generic Awesome Inc. theme.

I love the way Dynamic Views looks. It’s very sleek, but it’s impossible to adapt. After a dozen attempts I gave up getting Alex Gorbatchev’s SyntaxHighlighter to work with it, and a brief experiment with altering the CSS ended very badly indeed - it looked like a clown.

So to solve the first of those two problems I have switched to an old-style theme temporarily, while I write my own. This is not how I’d planned on spending the rest of my day, but I suppose my CSS skills could benefit from some practice. So look out for some new shiny, coming soon to a blog near you…
