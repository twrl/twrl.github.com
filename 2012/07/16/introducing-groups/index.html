Introducing Groups

Monday 16 July, 2012

Mathematics

For something a little different, I thought I’d write a little article on something from mathematics. This is partly just to test out the MathJax support on the new setup, but I am a maths student after all. My boyfriend - and my mum, and my housemates, and in fact most of the people in my life - don’t have a clue what I’m going on about half the time, so here’s an attempt at an explanation of one of my favourite mathematical concepts: Groups.

What is a Group?

A group is a set with an operation, which satisfies certain special properties.

So the first thing is to know what a set is. Well duh you say, a set is a collection of objects. This is true, but a set is a collection of objects without regard to duplicates. What that means is that a specific object can only be counted once. So if I have a set like this and one like this they are in fact the same set.

But what is an operation? Well, in this context an operation is a way of combining two elements of our group to get a third element. That’s not the proper definition, but it will do for now. We just write the elements out in order, so and so on. This mostly saves ink.

But what about these special properties. Well, the set has to have an element which we’ll write e, so that for every possible element x in the set We call e the identity element. It also has to be the case that every possible combination of two elements in the group is also an element in the group.
